Dieses **6sense Company Information Dataset Sample** umfasst **1.886 Einträge** und wurde mithilfe der **Bright Data API** extrahiert.

## Datenpunkte

| Feldname | Beschreibung |
|------------|-------------|
| `name` | Der offizielle Name des Unternehmens. |
| `about` | Eine kurze Beschreibung der Mission oder Dienstleistungen des Unternehmens. |
| `num_employees` | Die geschätzte Anzahl der Mitarbeiter, die im Unternehmen arbeiten. |
| `type` | Die Klassifizierung des Unternehmens, z. B. privat oder börsennotiert. |
| `industries` | Die Branchen, in denen das Unternehmen tätig ist, in einem Array-Format aufgeführt. |
| `techstack_arr` | Ein Array von Technologien und Tools, die vom Unternehmen genutzt werden. |
| `country_code` | Der ISO-Ländercode, der den Hauptsitz des Unternehmens repräsentiert. |
| `website` | Die offizielle Website-URL des Unternehmens. |
| `social_media_urls` | URLs zu den Social-Media-Profilen des Unternehmens. |
| `company_news` | Aktuelle Nachrichtenartikel oder Updates zum Unternehmen. |
| `last_updated` | Das Datum und die Uhrzeit, zu denen die Unternehmensinformationen zuletzt aktualisiert wurden. |
| `url` | Die URL, die zum Profil des Unternehmens auf der 6sense-Plattform führt. |
| `logo` | Die URL zum offiziellen Logo-Bild des Unternehmens. |
| `location` | Die vollständige Adresse des Hauptsitzes des Unternehmens. |
| `region` | Die geografische Region, in der sich das Unternehmen befindet, z. B. NA oder EU. |
| `id` | Eine eindeutige Kennung für das Unternehmen innerhalb des Datensatzes. |
| `slintel_resources` | Zusätzliche Ressourcen oder Referenzen von Slintel zum Unternehmen. |
| `stock_symbol` | Das Börsenkürzel des Unternehmens, falls zutreffend. |


## Anwendungsfälle

### 1. Marktforschung und Analyse
Nutzen Sie umfassende Unternehmensprofile, um Marktforschung und Wettbewerbsanalysen durchzuführen und Trends sowie Chancen in verschiedenen Branchen zu identifizieren.

### 2. Lead-Generierung und Outreach
Nutzen Sie detaillierte Unternehmensdaten, um Strategien zur Lead-Generierung zu verbessern und gezielte Outreach-Kampagnen zu entwickeln, wodurch die Conversion-Raten steigen.

### 3. Business Intelligence und Strategie
Integrieren Sie Unternehmensinformationen in Business-Intelligence-Tools, um strategische Erkenntnisse zu gewinnen, Abläufe zu optimieren und Wachstum voranzutreiben.
