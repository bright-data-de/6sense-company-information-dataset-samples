Dieses **6sense Company Information Dataset Sample** umfasst **1.886 Datensätze** und wurde mithilfe der **Bright Data API** extrahiert.

## Data Points

| Field Name | Description |
|------------|-------------|
| `name` | Der offizielle Name des Unternehmens. |
| `about` | Eine kurze Beschreibung der Mission oder Dienstleistungen des Unternehmens. |
| `num_employees` | Die geschätzte Anzahl der beim Unternehmen beschäftigten Mitarbeitenden. |
| `type` | Die Klassifizierung des Unternehmens, z. B. privat oder börsennotiert. |
| `industries` | Die Branchen, in denen das Unternehmen tätig ist, in einem Array-Format aufgeführt. |
| `techstack_arr` | Ein Array von Technologien und Tools, die vom Unternehmen genutzt werden. |
| `country_code` | Der ISO-Ländercode, der den Standort des Unternehmenssitzes repräsentiert. |
| `website` | Die URL der offiziellen Website des Unternehmens. |
| `social_media_urls` | URLs zu den Social-Media-Profilen des Unternehmens. |
| `company_news` | Aktuelle Nachrichtenartikel oder Updates im Zusammenhang mit dem Unternehmen. |
| `last_updated` | Datum und Uhrzeit der letzten Aktualisierung der Unternehmensinformationen. |
| `url` | Die URL, die zum Unternehmensprofil auf der 6sense-Plattform führt. |
| `logo` | Die URL zum offiziellen Logo-Bild des Unternehmens. |
| `location` | Die vollständige Adresse des Unternehmenssitzes. |
| `region` | Die geografische Region, in der sich das Unternehmen befindet, z. B. NA oder EU. |
| `id` | Ein eindeutiger Identifikator für das Unternehmen innerhalb des Datensatzes. |
| `slintel_resources` | Zusätzliche Ressourcen oder Referenzen von Slintel im Zusammenhang mit dem Unternehmen. |
| `stock_symbol` | Das Börsenkürzel, das das Unternehmen repräsentiert, falls zutreffend. |


## Use Cases

### 1. Marktforschung und Analyse
Nutzen Sie umfassende Unternehmensprofile, um Marktforschung und Wettbewerbsanalysen durchzuführen und Trends sowie Chancen in verschiedenen Branchen zu identifizieren.

### 2. Lead-Generierung und Outreach
Nutzen Sie detaillierte Unternehmensdaten, um Strategien zur Lead-Generierung zu verbessern und gezielte Outreach-Kampagnen zu konzipieren, wodurch die Conversion-Raten steigen.

### 3. Business Intelligence und Strategie
Integrieren Sie Unternehmensinformationen in Business-Intelligence-Tools, um strategische Erkenntnisse zu entwickeln, Abläufe zu optimieren und Wachstum voranzutreiben.
